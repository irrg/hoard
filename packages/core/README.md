# @irrg/hoard-core

Shared utilities for the [@irrg/hoard](https://github.com/irrg/hoard) provider packages.

Exports the `Scheduler` concurrency limiter, `fetchWithRetry`, `streamToFile`, `md5sum`, `runConcurrently`, and `NoDownloadError`. Used at runtime by `@irrg/drivethru-hoard`, `@irrg/itchio-hoard`, `@irrg/humblebundle-hoard`, `@irrg/bundleofholding-hoard`, and `@irrg/hoard`.
